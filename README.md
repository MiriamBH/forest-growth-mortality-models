# tree-growth-and-mortality-models
In this repository are the data and codes to reproduce the results of "Contrasting trends in forest growth and mortality of major European tree species under increasing climatic stress". 

Run the scripts in 02_analyses in ascending sequential order (from 01 to 08) to reproduce the results. 
Initial database named species_list.rds, needed to run all scripts, is included in 01_data. 
Script 05 needs aditional data (available online, but highly heavy in GB) to run, so the output database (cmi_sp_distribution.rds) is also included in 01_data folder.
